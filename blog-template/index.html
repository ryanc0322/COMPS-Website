Project Overview

Our project focuses on embedding social and civic responsibilities into Introduction to Computer Science. To do this, we emphasized real-world connections to the four main technical concepts that we chose to focus on and asked students to reflect upon these real-life examples throughout.


The four concepts we included are basics, loops and conditionals, dataset manipulation, and functions. Each concept had the following three components: take-home assignment, testing suite for the homework, and an in-class assignment. The homework portion includes a write-up, student-facing code, and the solution codes. The testing suite has a pytest and pylint file that grades the given code both on correctness and on style. The in-class assignment each includes a week's worth of material that can range from lab codes and presentation slides.
